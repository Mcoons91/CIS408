Volunteer Hours

Application to Fill In and Calculate Volunteer Hours

Password Login

About

This application is designed to calculate church volunteer hours still needed. Once the volunteer inputs their information, total hours still needed will be calculated. The formula is simple hours worked - total hours expected = hours needed. The volunteer can then view results of all volunteers hours on a graph beneath the form. This graph does not display calculated results however, it simply informational for all volunteers.

Additionally, the application will consist of a menu and password to login in with before beginning the application. The information entered by the user will be saved and displayed. The default password is set to 1717.

Instructions for App Usage

Please see the list of instructions below:

  1. Open the application using the internet browser icon of the page you are selecting.
  2. The first page is the homepage with these instructions and options for selection. Click a link to continue.
  3. The default password is set to 1717.
  4. This will take you to the Volunteer Hours application for user input.
  5. Results are saved and displayed for the user to view.
  6. Click either the graph option to view the graph or home to return to the homepage.
